## ¿Qué es y qué hace la práctica? / ¿Qué aprendimos?
# ¿Qué es y qué hace?
Es un detector de spam que analiza los mensajes que se envían a través de email y los clasifica como spam o no spam. Es muy básico pero tiene interfaz gráfica (muejejejej).

# ¿Qué aprendí?
Aprendí a usar de manera básica Git, GitHub, cómo crear entornos virtuales, cómo instalar librerías y, lo más importante, que Linus Torvalds también creó Git. Más que nada sería eso, pero también los conceptos básicos de Tkinter. Aunque de Tkinter falta bastante práctica, me hubiera gustado hacer el detector de spam paso a paso en vez de solo copiar y pegar el código ya hecho, siento que no aprendí mucho :(

## ¿Cómo lo hicimos? / Tecnologías-Librerías utilizadas
# ¿Cómo lo hicimos?
Con Python y Tkinter, siguiendo las instrucciones de las presentaciones y las maravillosas explicaciones del profesor Roberto y la doctora Paty.

# ¿Qué usamos para realizarlo?
Usamos Python, Tkinter e importamos librerías como scikit-learn, pandas, numpy. Utilizamos un archivo CSV como base de datos y con la colección de herramientas como Git y Python instalados (muejejejeje). "No sé por qué este salió con una sangría y los demás no".
